# Interaction

1. A participant stands in front of the camera.
2. The system detects one stable face for three seconds.
3. Facial data is translated into a unique synthetic cell.
4. The participant speaks (or remains silent) for several seconds.
5. Audio analysis influences the cell's behaviour, movement and activity.
6. The cell enters the virtual petri dish and continues interacting with all existing cells.

Every participant leaves behind a digital organism that remains active within the collective ecosystem.
